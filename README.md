# Employee-Attrition-Prediction-Using-Machine-Learning
Machine Learning project for predicting employee attrition using HR analytics with EDA, feature engineering, model comparison, and business insights.

### Dataset Description

The **IBM HR Analytics Employee Attrition & Performance Dataset** is a widely used HR analytics dataset for predicting employee attrition. It contains information about **1,470 employees** and **35 features**, including demographic details, job-related information, salary, work experience, job satisfaction, work-life balance, overtime, and performance metrics. The target variable is **Attrition**, which indicates whether an employee **left the company (Yes)** or **stayed (No)**.

The dataset includes features such as **Age, Gender, Department, Job Role, Monthly Income, Education, Marital Status, Years at Company, Job Satisfaction, Environment Satisfaction, Business Travel, Overtime,** and **Work-Life Balance**. These attributes help analyze the factors influencing employee turnover and build predictive models.

The primary objective of this dataset is to identify employees who are at risk of leaving the organization. By applying machine learning techniques, organizations can understand the key drivers of attrition and develop effective employee retention strategies, improve workplace satisfaction, and reduce turnover costs.
